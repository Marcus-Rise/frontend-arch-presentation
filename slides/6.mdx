# Docker

-  Развертывание проекта одной командой и автоматическая установка зависимостей
-  Чистая хост система и изолированная среда выполнения
-  Развертывание backend и базы данных, не собирая их, и возможность конфигурации сервиса

```yml
version: "3.4"

services:
  client:
    build:
      target: dev
    volumes:
      - .:/app
    ports:
      - 3000:3000
    env:
      - API_HOST:'backend:8080'

  backend
    image: registry.gitlab.com/user/project/backend:version
    env:
      - DB_HOST:'db:3306'
  db
    image: registry.gitlab.com/user/project/db:version
```
